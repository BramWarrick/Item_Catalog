Swiss Tournament

PostGreSQL and Python

PostGreSQL project with fully normalized datastructure, keys and views. Using a .sql file, all tables, keys and views were created, while the Python code allowed important code to be triggered. Tournament is Swiss-system, allowing all contestants to play in every round (byes excluded). Matches attempt to find the most equal opponent for every player. After the appropriate number of rounds, a winner is found. Two players will not rematch within a tournament, even if their skills and points would otherwise cause a pair.

Run tournament.sql then tournament.py will yield results.

Quiz - Fill in the Blanks

Project involved creating a quiz with multiple difficulty levels, user input and multiple attempts to answer correctly. I took the project further by adding color to the quiz entries and also by allowing multiple answers for a question - one correct answer with a few alternative approved answers being allowed. A prompt would inform the quiz-taker of the ideal answer, if it was not provided.

Run fill-in-the-blanks.py to initiate the quiz. Demonstrates loops, strings, user input and mark up.
